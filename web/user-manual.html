📖 Mini Cycle - User Manual

🚀 Getting Started

Welcome to Mini Cycle! This user manual will guide you through the features and functionalities of the app.

Mini Cycle is a new way of handling your repeatable tasks. Most task managers treat each task as a one-off item and collect your personal data. Mini Cycle is different - it's a privacy-focused workflow tool that organizes your tasks into manageable cycles. When you complete all your tasks, the cycle is completed and they automatically reset so you can tackle them again. It's perfect for daily routines, weekly goals, and recurring habits.

  • Click the + Add Task button to create your first task
  • Check off tasks as you complete them
  • When all tasks are completed, the cycle automatically resets (if Auto-Reset is enabled)
  • Use the slide arrows (❯❮) to switch between Task View and Stats Panel

📱 Navigation & Interface

🏠 Main Views

  • Task View: Your main task list and controls
  • Stats Panel: Analytics, milestones, and performance tracking

📍 Navigation Dots

The dots at the bottom show which view you're currently on and let you switch between them with a tap.

📱 Mobile Controls

  • Swipe left/right: Switch between Task View and Stats Panel
  • Long press task: Access task options (edit, delete, priority)
  • Drag tasks: Reorder your task list

✅ Managing Tasks

➕ Adding Tasks

Click "Add Task" and enter your task name. New tasks appear at the bottom of your list.

🔄 Reordering Tasks

Drag and drop tasks to reorder them. Your preferred order is saved automatically.

⭐ High Priority Tasks

Mark important tasks with the priority button. Priority tasks stand out visually to help you focus on what matters most.

📅 Due Dates & Times

  • Set due dates for time-sensitive tasks
  • Add specific times for precise scheduling
  • Overdue tasks are highlighted in red

🔔 Reminders

Enable reminders to get browser notifications about pending tasks. Make sure your browser allows notifications from Mini Cycle.

🔁 Recurring Tasks

Set up tasks that repeat automatically:

  • Daily: Resets every day
  • Weekly: Choose specific days of the week
  • Custom: Set your own interval

🔄 Cycle Management

⚙️ Auto-Reset

When enabled, all tasks automatically reset to incomplete once you finish everything. This is perfect for daily routines and recurring habits.

🎯 Manual Reset

Reset your cycle manually anytime using the reset button when you want a fresh start.

📋 Multiple Mini Cycles

Create different task lists for different areas of your life:

  • Work Tasks: Professional to-dos and projects
  • Home Routine: Household tasks and maintenance
  • Fitness Goals: Exercise routines and health habits
  • Study Plan: Learning objectives and skill development

🔄 Mini Cycle Switcher

Access the Mini Cycle switcher through the hamburger menu (☰) by clicking "Open":

  • Browse and select from your existing Mini Cycles
  • Preview cycle contents before switching - tasks appear in a preview window when you select a cycle
  • Rename or delete cycles using the action buttons (appear after selecting a cycle)
  • Click "Open" to switch to the selected cycle, or "Cancel" to close without switching

To create a new cycle, use the "Create New" button from the main menu, which will prompt you to name your new Mini Cycle.

To import .mcyc files from other devices or users, use the "Import Mini Cycle" option from the main menu.

Import .mcyc files to restore cycles or try out new setups created by others.

Clear all data and start completely fresh if needed. Note: This action cannot be undone, so make sure to backup anything important first.

❓ Frequently Asked Questions

How do I rename a Mini Cycle?

Open the Mini Cycle switcher and click the rename option next to your cycle name.

What happens if I delete a Mini Cycle?

Deleted cycles cannot be recovered unless you have an exported backup file.

Can I use Mini Cycle offline?

Yes! Mini Cycle works completely offline since all data is stored locally on your device.

How do I set up recurring tasks?

When editing a task, look for the recurring options to set daily, weekly, or custom intervals.

Why aren't my notifications working?

Ensure that the bell icon in the task options menu is enabled and that your device allows notifications from Mini Cycle.

How do I backup my data?

Use the export function in the Mini Cycle switcher to save .mcyc files as backups.

Can I sync data between devices?

Not automatically, but you can export from one device and import to another using .mcyc files.

What's the difference between priority and due dates?

Priority marks important tasks visually, while due dates create time-based reminders and warnings for deadlines.
