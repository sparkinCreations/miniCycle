## What is miniCycle?

miniCycle is a **routine-building task manager** that helps you establish consistent habits and processes. Unlike traditional to-do apps that focus on one-time tasks, miniCycle is designed for **repeating routines** that cycle.

### Key Differences from To-Do Apps

| Traditional To-Do App | miniCycle |
|----------------------|-----------|
| Tasks deleted when done | Tasks reset and repeat |
| Focus on one-time actions | Focus on routines and habits |
| Linear progress | Cyclical progress |
| Task list empties | Task list persists |

### Who Should Use miniCycle?

- **Routine-oriented people** who repeat the same tasks regularly
- **Process followers** who have checklists or procedures
- **Habit builders** establishing new routines
- **Teams** with recurring workflows
- Anyone who finds themselves recreating the same to-do list over and over

## Basic Concepts

### What is a Cycle?

A **cycle** is a named collection of tasks that represent a routine or process. Examples:
- "Morning Routine"
- "Weekly Cleaning"
- "Project Checklist"
- "Workout Plan"

### What Happens When You Complete a Cycle?

When all tasks in a cycle are checked off:
- ‚úÖ **Cycle count increments** (tracks how many times you've completed it)
- üîÑ **Tasks reset to unchecked** (ready for next time)
- üìä **Stats update** (completion rate, streaks)
- üéâ **Celebration animation** (optional)

This is the core "cycling" concept that makes miniCycle unique!

## Understanding Modes

miniCycle has **three operational modes** to match different workflows. **Mode changes apply instantly without page reload (v1.372+)** - your tasks update in real-time!

### 1. Auto Cycle Mode (Default)

**How it works:**
- Complete all tasks ‚Üí cycle **automatically resets**
- Best for: Daily routines, habits, recurring checklists

**Example:** Morning routine automatically resets every day when completed.

**Pros:**
- ‚úÖ Hands-free operation
- ‚úÖ Encourages consistency
- ‚úÖ Great for habits

**Cons:**
- ‚ö†Ô∏è Can't review completed state (resets immediately)

---

### 2. Manual Cycle Mode

**How it works:**
- Complete all tasks ‚Üí **"Complete Cycle" button appears**
- You manually click button to reset
- Best for: Weekly reviews, project phases, intentional resets

**Example:** Weekly cleaning checklist that you review before resetting.

**Pros:**
- ‚úÖ Control over when to reset
- ‚úÖ Can review completed state
- ‚úÖ Intentional completion

**Cons:**
- ‚ö†Ô∏è Requires manual action

**How to enable:**
1. Menu ‚Üí **Settings**
2. Find your cycle's settings
3. Toggle **"Manual Cycle Mode"**

---

### 3. To-Do Mode

**How it works:**
- Completed tasks are **deleted** (not reset)
- Best for: One-time tasks, traditional to-do lists

**Example:** Project tasks that won't repeat.

**Pros:**
- ‚úÖ Familiar to-do list behavior
- ‚úÖ List empties as you complete
- ‚úÖ Good for one-time projects

**Cons:**
- ‚ö†Ô∏è Tasks don't repeat (defeats cycling concept)

**How to enable:**
1. Menu ‚Üí **Settings**
2. Find your cycle's settings
3. Toggle **"To-Do Mode"**

## Recurring Tasks

Recurring tasks **automatically recreate** themselves on a schedule.

### When to Use Recurring Tasks

- ‚úÖ Tasks that repeat on a predictable schedule
- ‚úÖ Time-sensitive reminders
- ‚úÖ Maintenance tasks (daily, weekly, monthly)

**Examples:**
- "Take medication" (daily at 9 AM)
- "Water plants" (every 3 days)
- "Pay rent" (1st of every month)
- "Backup files" (every Sunday)

### Creating a Recurring Task

1. Add a task normally
2. Click **three dots (‚ãÆ)** ‚Üí **"Make Recurring"**
3. Choose schedule:
   - **Hourly** (e.g., every 2 hours)
   - **Daily** (e.g., every day at 8 AM)
   - **Weekly** (e.g., every Monday and Friday)
   - **Monthly** (e.g., 15th of every month)
   - **Yearly** (e.g., January 1st)
4. Set duration (forever, X times, or until date)
5. Click **"Save"**

### Understanding Recurring Behavior

**Scenario 1: Task Not Yet Due**
- Recurring task **won't appear** in your list yet
- Appears automatically when scheduled time arrives

**Scenario 2: Task is Due**
- Task appears in your list
- Check it off when complete
- Task disappears until next scheduled time

**Scenario 3: Missed a Task**
- If you miss a scheduled occurrence, task **still appears**
- Only one instance appears (not multiple for missed times)
- Check it off whenever you get to it

**Scenario 4: During Cycle Reset**
- Recurring tasks are **removed** during cycle reset
- They'll reappear at next scheduled time

### Editing Recurring Tasks

1. Click **three dots (‚ãÆ)** ‚Üí **"Edit Recurring"**
2. Modify schedule or duration
3. Click **"Save"**

Changes apply to **future occurrences** only.

### Stopping Recurring Tasks

1. Click **three dots (‚ãÆ)** ‚Üí **"Stop Recurring"**
2. Task becomes a regular task
3. Won't recreate anymore

## Tips & Best Practices

### üéØ For Daily Routines

1. **Order matters** - Put tasks in chronological order
2. **Start small** - Don't add too many tasks at first
3. **Be specific** - "Morning stretches (10 min)" better than "Exercise"
4. **Track time** - Add duration estimates to tasks
5. **Review weekly** - Adjust tasks that aren't working

---

### üîÑ For Recurring Workflows

1. **Use descriptive names** - "Weekly: Review analytics" better than "Check stats"
2. **Set reminders** - Don't rely on memory for scheduled tasks
3. **Batch similar tasks** - Group weekly tasks together
4. **Document steps** - Add sub-tasks for complex processes

---

### üìä For Habit Building

1. **Start with one habit** - Don't overwhelm yourself
2. **Make it easy** - Lower the barrier (e.g., "Read 1 page")
3. **Track streaks** - Use stats panel to see progress
4. **Celebrate wins** - Review cycle count regularly
5. **Adjust as needed** - Habits take time to stick

## Troubleshooting

### Tasks not saving

**Problem:** Tasks disappear after refresh

**Solutions:**
1. Check browser settings - ensure cookies/localStorage enabled
2. Try different browser
3. Check if private/incognito mode (data doesn't persist)
4. Export cycle as backup before troubleshooting

---

### Recurring tasks not appearing

**Problem:** Scheduled recurring task doesn't show up

**Solutions:**
1. Check system time - ensure clock is correct
2. Wait 30 seconds - recurring system polls every 30s
3. Refresh page - forces re-check
4. Verify schedule in task settings (Edit Recurring)

---

### App not updating

**Problem:** New features missing or old bugs persist

**Solutions:**
1. **Hard refresh:** Ctrl+Shift+R (Windows) or ‚åò+Shift+R (Mac)
2. **Clear cache:** Browser settings ‚Üí Clear browsing data
3. **Reinstall PWA:** Uninstall app ‚Üí Reinstall from browser
4. Check version number in Settings ‚Üí About

---

### Progress bar stuck

**Problem:** Progress bar doesn't update

**Solutions:**
1. Refresh page
2. Check if hidden tasks exist (expand completed dropdown)
3. Export and reimport cycle
4. Contact support if persists

---

### Mobile app not working offline

**Problem:** PWA requires internet

**Solutions:**
1. Open app while online first (downloads cache)
2. Check PWA installation - should be "installed" not just bookmarked
3. Check storage permissions
4. Reinstall PWA

---

### Lost data / Need to restore

**Problem:** Accidentally deleted cycle or tasks

**Solutions:**
1. **Undo button** - Click ‚Ü∂ (up to 20 undos per cycle)
2. **Import backup** - If you exported a `.mcyc` file
3. **Browser history** - Some browsers cache localStorage
4. **Prevention:** Export cycles regularly as backups!

### Common Questions

**Q: How many cycles can I create?**
A: Unlimited! Create as many as you need.

**Q: How many tasks per cycle?**
A: Recommended < 50 for performance. No hard limit.

**Q: Can I share cycles with others?**
A: Yes! Export as `.mcyc` file and share. Others can import it.

**Q: Does miniCycle sync across devices?**
A: Not built-in. Manual sync via export/import. Cloud sync may come in future.

**Q: Is my data private?**
A: Yes! All data stored locally in your browser. Nothing sent to servers.

**Q: Can I use miniCycle on multiple browsers?**
A: Yes, but data is per-browser. Use export/import to transfer.
