Quick Start Guide

Get up and running in 2 minutes:

  1. Launch miniCycle - Click the "Try Now" button to access the app
  2. Create your first miniCycle - Enter a name like "Daily Tasks" or "Morning Routine" when prompted
  3. Add your tasks - Type in the input box and click "Add" to create tasks
  4. Access task options - Hover over tasks (desktop) or long-press (mobile) to see edit, priority, and other options
  5. Set priorities - Use the priority button in task options to mark important tasks
  6. Complete tasks - Check off tasks as you finish them using the circular checkboxes
  7. Watch the magic - When all tasks are complete, they automatically reset for your next cycle (if Auto Reset is enabled)
  8. Explore features - Try the menu (☰) to access recurring tasks, themes, reminders, and more
